# Integrations

This folder contains skills for AI agents to use the YouTube Transcriber service.

## OpenClaw

```bash
cp -r contrib/openclaw ~/.openclaw/skills/youtube-transcriber
```

See [openclaw/README.md](./openclaw/README.md) for details.

## Claude Code

```bash
cp -r contrib/claude-code ~/.claude/skills/youtube-transcriber
```

See [claude-code/README.md](./claude-code/README.md) for details.

## ClawHub

The `clawhub/` directory contains the listing files for [ClawHub](https://clawhub.ai) — the OpenClaw skill directory. To submit or update the listing, zip the contents of `contrib/clawhub/` and upload at [clawhub.ai/import](https://clawhub.ai/import).

See [clawhub/README.md](./clawhub/README.md) for details.
